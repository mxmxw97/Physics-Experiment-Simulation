# 牛顿环虚拟仿真实验系统
## 需求规格说明书

> **文档版本**：V1.0
> **技术架构**：FastAPI + Three.js + 学生/教师双前端 + Excel 数据处理
> **部署方式**：教师电脑本地一键启动，局域网访问
> **运行环境**：Windows 10/11、Chrome/Edge 浏览器

---

## 一、项目概述
### 1.1 项目目标
构建**纯网页、免安装、局域网可用**的牛顿环虚拟仿真实验系统，实现全流程功能：3D 交互仿真、Excel 数据导入、智能校验纠错、自动计算、报告生成、批量批改、成绩统计，满足物理实验教学与竞赛要求。

### 1.2 核心约束
- 网页端实现 3D 交互
- 无数据库，采用文件化存储
- 教师电脑一键部署，学生浏览器直接访问
- 支持 30–50 人并发使用

### 1.3 用户角色
- **学生**：虚拟实验、数据上传、自动计算、报告提交
- **教师**：服务管理、批量批改、成绩统计、报表导出

---

## 二、系统总体架构
### 2.1 分层架构
- 前端层：学生端 `/student`、教师端 `/teacher`
- 接口层：FastAPI 提供 RESTful API
- 核心层：物理计算、数据校验、智能批改
- 存储层：本地文件系统（上传文件/报告/成绩/配置）

### 2.2 数据流向
1. 学生上传 Excel → 后端校验 → 自动计算 → 生成报告 → 提交存档
2. 教师获取提交列表 → 批量批改 → 成绩统计 → 报表导出

### 2.3 技术栈
- 后端：Python 3.9+、FastAPI、Uvicorn、Pandas、NumPy、OpenPyXL
- 前端：HTML/CSS/JS、Three.js
- 3D 模型：Blender 导出 glTF/GLB
- 部署：Windows 批处理脚本、PyInstaller 打包

## 三、学生端功能需求
### 3.1 3D 虚拟仿真模块
- 加载牛顿环实验装置 3D 模型（钠光灯、显微镜、平凸透镜、载物台、底座）
- 支持鼠标旋转、缩放、平移、部件点击高亮
- 可交互操作：目镜调焦、镜筒升降、反光镜调节、测微鼓轮旋转
- 参数可调：波长 λ、曲率半径 R、折射率 n
- 误差模拟：空程差、偏心误差、随机读数误差
- 实时渲染干涉条纹与光强分布曲线

### 3.2 实验数据导入模块
- 提供标准 Excel 模板下载
- 支持 `.xlsx` 文件上传与解析
- 模板列：环级数、左读数、右读数、直径
- 数据预览、格式校验、异常值提示

### 3.3 智能数据校验与纠错模块
#### 3.3.1 合法性校验
- 读数在 0–10 mm 量程内
- 左读数 ＜ 右读数
- 直径随环级数递增
- 直径在 0.1–5 mm 合理范围

#### 3.3.2 误差校验
- 直径突变粗差识别
- 空程差违规（反向读数）检测
- D²–k 线性拟合 R² < 0.99 标记异常

#### 3.3.3 纠错输出
- 错误行号、类型、原因、修正建议

### 3.4 自动数据处理模块
- 直径计算：`D = 右读数 − 左读数`
- 逐差法计算曲率半径 R
- 不确定度计算：A 类、B 类（0.004 mm）、合成不确定度、相对误差
- 输出完整计算过程与结果

### 3.5 实验报告与提交模块
- 网页端展示完整实验报告
- 支持导出 Excel/PDF 报告
- 一键提交至教师服务器

### 3.6 教学辅助模块
- 实验原理、公式、操作步骤、注意事项、常见错误说明

## 四、教师端功能需求
### 4.1 提交管理模块
- 展示学生提交列表：序号、学号/姓名、提交时间、状态、得分
- 支持查看详情、删除记录、清空全部、开启/关闭提交通道

### 4.2 批量自动批改模块
- 一键批量批改
- 按错误类型自动打分（100 分制）
- 错误统计：读数越界、左右反、空程差、数据突变、趋势错误、线性度低
- 自动生成批改评语

### 4.3 成绩与统计模块
- 班级统计：最高分、最低分、平均分、提交率、合格率
- 可视化：分数分布柱状图、错误类型饼图
- 支持手动修改分数、添加批注

### 4.4 报表导出模块
- 导出全班成绩单、批改汇总报表、单份报告、错误类型统计报表

### 4.5 系统管理模块
- 模板下载、批改规则配置、服务状态监控、数据备份/清空

## 七、接口详细设计
### 7.1 公共规范
- 基础路径：`http://{server-ip}:8000/api`
- 通用返回格式：
{
  "code": 200,
  "msg": "success",
  "data": {}
}
### 7.2 学生端接口
| 接口路径 | 请求方式 | 功能说明 |
| -------- | -------- | -------- |
| /student/upload | POST | 上传Excel实验数据文件 |
| /student/check | POST | 实验数据智能校验与纠错 |
| /student/calculate | POST | 自动计算曲率半径R与不确定度 |
| /student/submit | POST | 提交实验报告至服务器 |
| /student/template | GET | 下载标准实验数据Excel模板 |

### 7.3 教师端接口
| 接口路径 | 请求方式 | 功能说明 |
| -------- | -------- | -------- |
| /teacher/list | GET | 获取全班学生提交记录列表 |
| /teacher/correct | POST | 批量自动批改所有提交 |
| /teacher/stats | GET | 获取班级成绩与错误统计 |
| /teacher/export/scores | GET | 导出全班成绩单Excel |
| /teacher/export/report | GET | 导出批改汇总报表 |
| /teacher/score/edit | POST | 修改学生成绩与添加批注 |
| /teacher/delete | POST | 删除单条/批量提交记录 |
| /teacher/config/submit | POST | 开启/关闭学生提交通道 |

### 7.4 系统接口
| 接口路径 | 请求方式 | 功能说明 |
| -------- | -------- | -------- |
| /server/status | GET | 查询服务器运行状态 |
| /server/restart | POST | 重启后端服务 |

## 八、部署与交付
### 8.1 部署方式
- 一键启动脚本：`start_server.bat`（Windows双击运行）
- 一键停止脚本：`stop_server.bat`
- 服务启动后自动打开系统首页，展示学生端/教师端入口
- 局域网内通过 `http://教师电脑IP:8000` 访问

### 8.2 交付物清单
1. 绿色免安装运行包（内嵌Python环境）
2. 学生端 + 教师端完整前端静态文件
3. 标准实验数据Excel模板
4. 接口设计文档
5. 教师/学生使用说明书
6. 系统演示视频（≤10分钟）

### 8.3 运行环境要求
- 操作系统：Windows 10/11
- 内存容量：≥4GB
- 浏览器：Chrome、Edge最新版本
- 网络：班级设备与教师电脑处于同一局域网
